# ESTUDIO DE VIABILIDAD DEL PROYECTO

**Proyecto:** Agente de IA especializado en problemas del hogar\
**Autor:** Francisco José Cortés Carmona -- 2º DAW

## 1. Introducción

El proyecto consiste en desarrollar una aplicación web que integre un
agente inteligente basado en Gemini, capaz de ayudar a los usuarios a
resolver problemas cotidianos del hogar, como mantenimiento básico,
limpieza, organización doméstica y consultas sobre electrodomésticos. El
objetivo es ofrecer una herramienta accesible, intuitiva y útil mediante
una interfaz web donde el usuario interactúe a través de un chat.

## 2. Objetivos

### Objetivo general

Desarrollar una aplicación web funcional que incorpore un agente de IA
especializado en tareas y problemas comunes del hogar.

### Objetivos específicos

-   Crear un backend capaz de comunicarse con la API de Gemini.
-   Diseñar una interfaz web clara y fácil de usar.
-   Gestionar historiales o preferencias del usuario (si se requiere).
-   Garantizar la seguridad básica del sistema.
-   Aplicar conocimientos del ciclo 2DAW en frontend, backend y APIs.

## 4. Requisitos

### Requisitos funcionales

-   Enviar preguntas al agente de IA.
-   Conectarse a la API de Gemini y mostrar respuestas.
-   Interfaz clara y responsiva.
-   Backend seguro y estable.

### Requisitos no funcionales

-   Rapidez en la carga.
-   Compatibilidad móvil.
-   Código organizado y mantenible.

### Requisitos técnicos

-   Equipo capaz de ejecutar Python y/o Node.
-   Conexión a Internet para la API.
-   Hosting simple para despliegue.

## 5. Viabilidad

### Viabilidad técnica

Las tecnologías propuestas son compatibles, actuales y fáciles de
integrar. El procesamiento principal lo realiza Google Gemini, por lo
que la carga del sistema es baja. El proyecto es técnicamente viable.

### Viabilidad económica

El único coste sería el uso de la API de Gemini, pero existen planes
gratuitos o económicos. Viabilidad alta.

### Viabilidad operativa

El proyecto es adecuado al tiempo y nivel de un estudiante de 2DAW. Es
totalmente realizable.

### Conclusión

El proyecto es **viable**, tanto técnica como económicamente y
operativamente. Puede desarrollarse con garantías dentro del marco
académico de 2DAW.
